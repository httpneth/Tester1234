Temp Mail API Documentation

Base URL

https://temp-mail-api.codersensui.repl.co

Generate Random Email

Endpoint: /api/gen

Method: GET

Description: Generates a random email address.

Response:

{
  "email": "example@domain.com"
}

Get Messages for Email

Endpoint: /api/getmessage/:email

Method: GET

Description: Retrieves messages for a given email address.

Parameters:

  • :email (path parameter): The email address to retrieve messages for.

Response:

{
  "messages": [
    {
      "sender": "sender@example.com",
      "subject": "Example Subject",
      "message": "This is an example message."
    },
    {
      "sender": "another.sender@example.com",
      "subject": "Another Subject",
      "message": "This is another example message."
    }
  ]
}
